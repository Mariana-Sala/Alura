# CONVERSOR ALURA 
Repositorio del Challenge Conversor para Oracle Next Education 2023, realizado por Mariana Toro

# Descripcion
Este proyecto, es una aplicacion que proporciona la funcionalidad de cambiar divisas, temperatura y diferentes medidas de longitud. La aplicacion usa una API para obtener las tasas de cambio mas recientes, mientras que para temperatura y longitudes se implemento una sentencia if-else. 

La interfaz de usuario se desarrollo utilizando Java Swing.
